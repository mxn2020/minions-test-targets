# Minions Test-targets

Agent-under-test definitions, endpoints, capabilities, and auth configs

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-test-targets/sdk`) — Core library for Node.js
- **Python SDK** (`minions-test-targets`) — Python bindings
- **CLI Tool** (`@minions-test-targets/cli`) — Command-line interface
- **Web App** — Interactive playground at [test-targets.minions.wtf](https://test-targets.minions.wtf)
