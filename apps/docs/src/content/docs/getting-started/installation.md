---
title: Installation
description: Install Minions Test-targets packages
---

## TypeScript / Node.js

```bash
npm install @minions-test-targets/sdk minions-sdk
```

## Python

```bash
pip install minions-test-targets
```

## CLI

```bash
npm install -g @minions-test-targets/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
